# Miraust
Mirai Rust Interface

# 如何使用
### 1. 下载console插件并放入plugins文件夹内，启动一次mirai-console。

### 2. 将`miraust_core`库文件放入`mirai/miraust/core`目录下

### 3. 将插件库文件放入`mirai/miraust`目录下，开启mirai-console
